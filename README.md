# 🌊 Численное моделирование: Решатели вязкопластических течений

В этом репозитории собраны реализации численных методов для решения задач механики сплошных сред. Проекты разработаны в рамках научно-исследовательской работы на **Мехмате МГУ** (кафедра аэромеханики и газовой динамики).

Основной фокус — эффективные алгоритмы для моделирования течений сложных сред (например, вязкопластических жидкостей).

---

## 🚀 Основные проекты

### 📐 Течение в L-образной области
**Папка:** `Solver_L_ALG_FISTA`
*   **Задача:** Моделирование течения в канале сложной формы (L-образная область).
*   **Алгоритм:** Реализация **FISTA** (Fast Iterative Shrinkage-Thresholding Algorithm) для ускоренной минимизации функционала.
*   **Инструменты:** C++ с использованием многопоточности и оптимизацией разреженных матриц. Для визуализации полей скоростей и жестких зон используется Python.

### 🔳 Течение в каверне (Caverna)
**Папка:** `Solver_ALG_FISTA_caverna`
*   **Задача:** Моделирование 2D-течения в квадратной полости.
*   **Особенности:** Сравнение классического метода ALM (Augmented Lagrangian Method) и ускоренных вариантов на базе FISTA.
*   **Математика:** Высокоточные решатели уравнения Пуассона и итерационные методы обновления полей.

### 🧪 Прототипы и простые течения
**Папка:** `solve_ALG2`
*   **Задача:** Течения в прямолинейных каналах.
*   **Метод:** Реализация итерационного алгоритма **ALG2**, который обеспечивает стабильную сходимость для вязкопластических моделей.
